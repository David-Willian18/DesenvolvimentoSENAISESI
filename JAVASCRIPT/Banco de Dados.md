Um banco de dados, uma coleção estruturada de informações, onde são armazenadas de forma eletrônica para ter um acesso fácil, precisa ser gerenciada para que esses dados estejam seguros, íntegros e com desempenho. O gerenciamento de dados serve para auxiliar as organizações para otimizar o uso das informações, envolvendo criação, atualização e o acesso. Uma das formas de dessa prática são os bancos relacionais, que são sistemas de gerenciamento que organizam os dados em tableas com linhas e colunas, assim tendo relacionamentos lógicos entre elas por meio de chaves. Seguindo essa linha de raciocínio, esses dados podem ser representados de forma mais visual e flúido. Isso pode acontecer por meio do diagrama de dados, um tipo de fluxograma, que utiliza símbolos e formas para mostrarem como os dados estão sendo relacionados.
